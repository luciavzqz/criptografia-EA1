# Criptografía EA1
Evaluación de Aprendizaje 1 de la matería Criptografía - UNLaM

Armar equipos. Investigar, Desarrollar e implementar un programa para:

- Trabajar con números enteros grandes de diferentes tamaños, desde 10 dígitos (33 bits)
hasta 90 dígitos (297 bits), que permita realizar distintas operaciones.
- Un generador de números aleatorios (utilizando una semilla que no pueda predecirse).
- Un algoritmo que verifique si un número es primo.

*Tipo:* trabajo de investigación y desarrollo.

*Tiempo:* 1 semana

*Fecha:* 28/09/2020 al 05/10/2020. El 05/10 cada equipo mostrará el software funcionando
